Búsqueda de Patrones de Texto

En este tema, exploraremos cómo un autómata puede ser utilizado para buscar patrones dentro de un texto. Tomaremos como ejemplo las palabras "web" y "ebay".

Conversión de AFND a AFD

Un AFD (Autómata Finito Determinista) no permite ambigüedad: para cada estado y símbolo debe haber una única transición.

Pasos Básicos

Para convertir el AFN anterior en un AFD:

  • Se crean rutas únicas para cada palabra, desde el estado inicial.
  • No hay transiciones vacías ni múltiples transiciones por símbolo.
