Tema 9: Cálculo de Clausura

Definición de Clausura

La clausura-ε de un estado q, denotada como ε-Closure(q), es el conjunto de todos los estados a los que se puede llegar desde q usando únicamente transiciones vacías (ε).

Ejemplo de Cálculo de Clausura

Tomando como ejemplo el automata anterior, se detallan las clausuras para cada estado de este

  • ε-Closure(q0):
    • Desde q0, hay transiciones ε hacia q1 y q3.
    • Resultado: {q0, q1, q3}

  • ε-Closure(q1):
    • No hay transiciones ε desde q1.
    • Resultado: {q1}

  • ε-Closure(q2):
    • No hay transiciones ε desde q2.
    • Resultado: {q2}

  • ε-Closure(q3):
    • No hay transiciones ε desde q3.
    • Resultado: {q3}

  • ε-Closure(q4):
    • No hay transiciones ε desde q4.
    • Resultado: {q4}

  • ε-Closure(q5):
    • No hay transiciones ε desde q5.
